# Ads1256 Library Class
Driver for fast 8 channel 24bit ADC converter from TI

## Goals
* easy to port
* light weight
* full features
* easy to use

## Status
works for arduino framework on ESP32, reading
* single raw values
* raw values in bulk
* swipe across channels
